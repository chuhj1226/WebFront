자손/후손 선택자

자손 : 바로 아래의 요소 선택, 후손 : 하위 요소 전부 

 선택자 > 자손 선택자 { 설정 내용 } 
선택자 후손 선택자 { 설정 내용 } 의 형식으로 사용

등위 선택자

등위 관계에서 뒤에 위치한 태그를 선택할 때 사용한다 

- 바로 뒤의 요소 선택 시 : 선택자a + 선택자b { 설정 내용 },
- 뒤에 있는 모두를 선택 시 : 선택자a ~ 선택자b { 설정 내용} 의 형식으로 사용

반응 선택자

사용자의 움직임에 따라 달라지는 선택자 

사용자가 클릭한 태그 : 선택자:active { 설정 내용 },
사용자의 마우스가 올라가 있는 태그 : 선택자:hover { 설정 내용 } 의 형식으로 사용

hover 테스트

상태 선택자

입력 양식의 상태에 따라 선택 되는 선택자 

체크 상태의 input 태그 선택 : input태그:checked { 설정 내용 } 의 형식으로 사용
초점이 맞추어진 input 태그 선택 : input태그:focus { 설정 내용 } 의 형식으로 사용


사용 가능한 input 태그 선택 : input태그:enabled { 설정 내용 },
사용 불가능한 input 태그 선택 : input태그:disable { 설정 내용 } 의 형식으로 사용
