JQuery 개요

JQuery란?

기존 복잡했던 클라이언트 측 HTML 스크립팅을 간소화 하기위해 고안된 JavaScript 라이브러리로 적은 양의 코드를 빠르고, 풍부한 기능을 제공함

JQuery 연결 방법

    - head 태그 내부에 script 태그 선언
    <script type="text/javaScript" src="js" 파일경로.js" >
    </script>

    오프라인 / 온라인 상태에 따라 파일 경로 저장

    - JQuery 공식 사이트 : https://jquery.com
    - jquery 라이브러리 제공 페이지 : https://code.jquery.com

    1. JQuery 라이브러리 다운로드를 통한 연결
        - js 파일을 다운로드 하여 파일 경로 지정
    2. CDN(Content Deliver Network)를 통한 연결
        - 제공하는 주소를 이용하여 파일 경로 지정

jQuery 시작하기

    - 문서가 로드 된 후 실행하는 방법 3가지

    1) jQuery(document).ready(function(){});
    2) $(document).ready(function(){});
    3) $(function(){});

    - jQuery 기본 형태
    $('선택자').메소드명('속성','속성값');
