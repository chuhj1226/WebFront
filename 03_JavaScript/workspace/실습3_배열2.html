문제 5

아래 선언 된 변수를 "," 기준으로 분리하여 선언 된 순서대로 하나의 배열로 결합한다.
prompt로 사용자에게 입력 받은 값이 해당 배열에 존재하면 alert를 통하여 배열 내 위치를 출력하고 존재하지 않으면 alert로 존재하지 않습니다를 출력하는 findWord 함수를 완성한다.
단, prompt 창에서 취소 또는 입력 값 없이 확인을 눌렀을 경우는 결과를 확인할 수 없습니다를 alert를 통하여 출력한다.

Ex. 사용자 prompt 입력 값 : 사과, alert 결과 값 : 사과는 배열 index 0번째에 있습니다.
Ex. 사용자 prompt 입력 값 : 강아지, alert 결과 값 : 강아지는 배열 index 11번째에 있습니다.
Ex. 사용자 prompt 입력 값 : 햄스터, alert 결과 값 : 햄스터는 배열에 존재하지 않습니다.
Ex. 사용자 prompt 입력 시 취소 또는 빈칸 입력, alert 결과 값 : 결과를 확인할 수 없습니다.
