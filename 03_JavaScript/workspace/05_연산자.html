연산자

기본적인 연산자의 사용과 우선 순위 등은 Java와 크게 다르지 않으므로 차이가 있는 부분에 대해서만 확인

비교 연산자를 통한 문자열 비교

비교 연산자 (크다, 작다, 작거나 같다, 크거나 같다)로 문자열을 비교할 수 있음
사전 순(유니코드 순)으로 뒤쪽 문자열은 앞쪽의 문자열보다 크다

==, != 연산자와 ===, !== 연산자

동등 연산자(==, !=)

자료형에 상관 없이 값이 일치하면 true, 아니면 false

일치 연산자(===, !==)

자료형과 값이 모두 일치하면 ture, 아니면 false

그 외 삼항 연산자, 조건문(if문, if-else문, if-else if문, switch문). 반복문(for문, while문, do-while문) 등의 사용 방법 모두 Java와 동일함
