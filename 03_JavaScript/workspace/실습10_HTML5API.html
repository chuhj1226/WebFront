문제 1

아래 div 영역을 클릭 시 파일 첨부창이 뜨고 이미지 파일 첨부 시 해당 파일을 div에 프리뷰로 표현하는 코드를 작성한다.

[참고]
document.querySelectorAll(".imgUpload")[index], document.querySelectorAll(".imgUpload").forEach() 등은 가능하지만
document.querySelectorAll()의 리턴 값은 NodeList로 유사 배열이기 때문에 배열에서 사용하는 기능은 사용할 수 없다.
배열의 메소드를 사용하고 싶다면 해당 NodeList를 Array.from() 메소드를 통해 배열로 변경해서 사용한다.
Ex. Array.from(document.querySelectorAll(".imgUpload")).indexOf()

문제 2

블루 체육관과 노랑 체육관에 포켓몬들이 있다. 각 체육관을 자유롭게 이동할 수 있도록 drag and drop 기능을 구현한다.
div 영역에 drop 될 경우 가장 마지막에 추가하고 img 영역에 drop 될 경우 해당 이미지 다음 차례로 추가한다.

[참고]
1. 시작점에서 드래그가 시작 되는 순간 해당 타겟의 아이디를 저장한다.
2. HTML 요소들의 기본 값은 드롭을 받아 들이지 않게 되어 있음을 고려한다.
3. 도착점에서 드롭이 발생했을 때 div에 드롭이 발생했는지 img에 drop이 발생했는지 확인해서 올바른 위치에 삽입한다.
