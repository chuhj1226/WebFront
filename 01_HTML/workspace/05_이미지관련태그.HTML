이미지 관련 태그

scr 속성

scr 속성에 경로를 설정하여 이미지 파일을 불러옴

alt 속성

사진의 경로가 잘못 되거나 이미지를 제대로 표현할 수 없는 경우 대체 텍스트의 용도로 사용
또는 시각 장애인을 위한 화면 낭독기 프로그램에서 이미지를 대체하는 텍스트를 작성

width와 height

사진의 높이와 너비를 지정할 수 있다
고정 된 단위(px 등) 의 크기로도 지정 가능하며 가변 크기 단위(%)의 크기로도 지정 가능하다

고정 크기 단위 : 화면 사이즈가 줄어도 크기가 변하지 않음

가변 크기 단위 : 화면 사이즈 혹은 기준 사이즈에 따라 크기가 변경

이미지 구역을 2개로 나누어 각각 링크 설정

map, area : 이미지를 구역별로 나누어 링크를 걸어주는 태그 - img 태그에 usemap 속성 지정 - shape : 링크로 사용할 영역의 형태 지정(직사각형, 원형) - coords : 분할할 지역 좌표 지정(시작,끝) - href : 링크 주소 지정 - target : 링크 페이지가 열릴 위치(새창, 현재창)
