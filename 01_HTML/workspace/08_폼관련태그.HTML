폼 관련 태그

form 태그는 html에서 사용자가 입력할 수 있는 양식을 제공하는 태그
form 태그 내의 input 태그들을 통해 사용자가 입력한 정보를 서버로 넘기는 역할을 함

submit 버튼을 눌렀을 때 action에 지정 된 경로, method에 지정한 방식으로 input 태그의 값을 전달함
* action : 폼 내부에 입력 된 사용자 입력 양식의 값을 전송 받을 서버의 주소 입력
* method : get/post 방식으로 전속 방식 지정
설정하지 않았을 경우 get 방식으로 지정(url창으로 데이터를 보내는 방식)
post 방식으로 지정하면 url 창에 데이터가 보이지 않음

fieldset : 폼 요소를 그룹으로 묶는 태그

legend : 묶은 폼 요소에 명칭을 붙이는 태그
