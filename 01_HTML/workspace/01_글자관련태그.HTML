문단을 나누는 태그(p, pre)

문단 영역을 나누는 태그로는 p태그와 pre태그가 있다.

p태그는 문단 영역을 나누는 태그이지만 한 개의 공백만 표시하며
줄 바꿈 입력을 별도의 태그로 지정해 주어야 한다 

pre태그는 여러 칸           띄우기
    혹은 줄 바꿈 등을 포함하여
    입력한 내용 그대로 표현하는 태그이다

일반 글꼴
b : 글자를 굵게 표시하는 태그
strong : 글자를 굵게 표시하는 태그
i : 글자를 기울이는 태그
em : 글자를 기울이는 태그
blockquote : 인용 문구를 나타내는 태그
- cite 속성 : 인용 된 사이트 주소를 작성
직접 표시되진 않지만 검색 엔진에는 노출 됨
q: 인용 문구를 나타내는 태그
mark : 형광팬 효과를 나타내는 태그
u : 글자에 밑줄을 긋는 태그
small : 글자를 작게 표시하는 태그
기본 글자에 아래 첨자를 나타내는 sub태그와 sup : 윗 첨자를 나타내는 sup태그
s: 글자에 취소선을 넣는 태그
abbr : 마우스 오버 시 툴팁 형태로 출력하는 태그

cite : 참고한 웹 문서를 이탤릭체로 나타내는 태그
